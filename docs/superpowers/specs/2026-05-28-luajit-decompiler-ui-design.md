# LuaJIT Decompiler UI 2.0 设计规范

日期：2026-05-28

## 1. 概述

对 LuaJIT 在线反编译器进行 UI 现代化改造，采用 macOS Sonoma 设计语言，提升用户体验。

## 2. 视觉风格

### 设计系统
- **风格**：macOS Sonoma - 毛玻璃效果 + 圆角卡片 + 半透明背景
- **主色调**：蓝紫渐变 `#667eea → #764ba2`
- **背景色**：`#f8fafc` 浅灰白
- **字体**：SF Pro Display / Inter / system-ui
- **圆角**：16-24px（大卡片）、8-12px（按钮）、4-6px（输入框）

### 毛玻璃效果
```css
backdrop-filter: blur(20px) saturate(180%);
background: rgba(255, 255, 255, 0.72);
border: 1px solid rgba(255, 255, 255, 0.3);
```

## 5. 文件类型智能识别

| 文件类型 | 检测方式 | 处理方式 |
|---------|---------|---------|
| Lua 源码 (.lua) | 扩展名 / magic bytes | 显示文本预览 |
| Lua Bytecode (.luac) | 扩展名 / magic bytes `\033Lua` | 反编译，显示 Lua |
| 文本文件 | UTF-8/ASCII 检测 | 显示文本预览（只读 Monaco） |
| 音视频 (mp3/mp4/wav/avi) | magic bytes | 内嵌播放器 |
| 图片 (jpg/png/gif/webp) | magic bytes | 显示缩略图预览 |
| 其他二进制 | magic bytes | 警告提示 + 文件大小 |

### Magic Bytes 参考
```
Lua Bytecode: 1B 4C 75 61 (ESC + "Lua")
JPEG: FF D8 FF
PNG: 89 50 4E 47
MP3: 49 44 33 或 FF FB
MP4: 00 00 00 XX 66 74 79 70
```

## 6. 响应式断点

- **Desktop** (≥1024px)：完整三栏布局
- **Tablet** (768-1023px)：侧边栏可折叠
- **Mobile** (<768px)：单栏 + 底部抽屉
